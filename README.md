# PLY_to_USD
Convert gaussian splatting PLY file into USD file which be used in NVIDIA Omniverse.

- Tested python version: python 3.10
- Run `pip install -r requirements.txt` to install the packages needed for this program (recommend using conda)
- Convert existing PLY file to USD: run this command in cmd `python ply_to_usdz_converter.py model.ply` (replace model.ply with your file name)
